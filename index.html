FAQ

How to add more characters in Svarog database?

1. change the in-game character showcase or character support

2. use the /profile user slash command
Is there a rate limit for HSR UID lookup requests?

10 minutes/uid, it will be stored in the cache temporarily for related command responses in 10 minutes
it possible to register more than one HSR UID for each Discord user?

No (a unique user ID), if you register another UID it will delete all data on your old UID.
Will the leaderboard message always be updated?

The leaderboard keeps updating every time there is a button interaction. so, use lock button to make the button always active.
Is there a risk of being banned from using this service?

No, we are using public Mihoyo API the same requests as in-game to get your data. There is no risk of being banned by using our services.
Why is my HSR data shown inaccurately as in-game?

Bot only syncs your game data to the svarog database when you or someone requests a profile lookup on your UID.
What local & global leaderboard mean?

"Local" list of users from all Discord servers.
"Global" list of users in your Discord server only
How to hide my HSR UID from public view?
